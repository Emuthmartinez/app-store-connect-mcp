Why StorePilot

60+ tools, one vocabulary

Read listings, update keywords, manage screenshots, run Custom Product Pages and Product Page Optimization experiments, submit versions for review — all as MCP tools your agent can call.

Revenue correlation, built in

Every listing mutation is logged with the RevenueCat metrics at the time of change. asc_get_change_impact_analysis tells you which edits preceded subscription lifts.

Scheduled health reports

Weekly markdown digest of your listing score, revenue trend, and recent change impact — piped to email or Slack, all automated.

Slack & Discord alerts

Get pinged the moment MRR shifts, health scores drop, or a review state changes. One webhook URL, zero glue code.

Open source & self-hostable

Run it yourself for free under MIT. Clone the repo, run python3 src/index.py, and you're in business. Cloud tier is convenience, not gatekeeping.

Audit-ready mutation log

Every write emits a JSONL record with before/after diff and the metrics snapshot at mutation time. Enterprise-grade audit trail from day one.
